# Data dictionary

trip_id
- Unique identifier for each trip

start_time / end_time
- Timestamp of trip start and end (YYYY-MM-DD HH:MM)

bike_id
- Identifier of the bike used

trip_duration_minutes
- Duration in minutes (numeric)

from_station_name / to_station_name
- Station names for origin and destination

member_type
- Subscriber (member) or Customer (casual)
