# Cyclistic Bike Share Analysis

Short description
This repository contains a reproducible spreadsheet-based analysis of the Cyclistic bike share dataset. The analysis is designed to be done primarily with Excel (or other spreadsheet software) and includes sample data, a data dictionary, and step-by-step analysis instructions.

Contents
- data/ — sample CSV files you can open in Excel
- analysis/ — analysis plan, steps and example calculations
- docs/ — supplementary notes
- .github/ — issue templates and contribution guidelines

Getting started (Excel)
1. Download or clone the repo.
2. Open files in Excel (CSV files can be opened directly or imported).
3. Follow steps in analysis/analysis_plan.md to reproduce analyses and charts.

Suggested analysis questions
- Ridership trends by month/day/hour
- Trip duration distribution and summary statistics
- Most popular start/end stations
- Differences in usage between member types
